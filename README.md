# Getting Started with React Project

## Set Up Environment

1. **Node.js and npm:**

   - Install Node.js and npm from the [official website](https://nodejs.org/).

2. **Install package:**

   ```bash
   npm install

   ```

3. **Run the App:**
   ```bash
   npm start
   ```
